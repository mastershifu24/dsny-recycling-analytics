DSNY Tonnage Voice

Uses DSNY Monthly Tonnage open data: refuse / recycling tons by month, borough, and community district. Ask for trends, shift / month over month, or name a borough (e.g. Bronx). Set NYC_SODA_DATASET=c23c-uwsm on the server to use SweepNYC instead.

Dataset: DSNY Monthly Tonnage (ebb7-mvp5)
