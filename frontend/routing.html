Rank & optimize garbage truck routes

Goal: ordered stops using road driving times (OSRM) when possible, plus datasets for constraints. Borough tonnage in the main app is separate.

Basic stop optimizer (built-in)

Paste JSON: an array of stops with lat, lng, optional label. First stop is the default start. Uses nearest-neighbor on OSRM driving durations (falls back to straight-line time if OSRM is down).

DOT — truck route network (constraints for driving)
  • New York City Truck Routes (Map) — NYC Open Data

Use this layer so optimized paths respect where commercial trucks may or must run.

Street graph (for routing engines)
  • NYC Street Centerline (CSCL)

OSRM above uses OpenStreetMap; full VRP can add time windows and capacity later.

Pickup day (per address)
  • DSNY collection schedule lookup

API: POST /api/route/optimize with body {"stops":[...], "start_index":0, "use_osrm":true}
